CARA PAKAI CEPAT

1. Buka Supabase > SQL Editor.
2. Copy semua isi file supabase_sql_editor.sql lalu klik RUN.
3. Upload index.html ke GitHub pada root repository.
4. Deploy repository itu ke Vercel.
5. Setelah website terbuka, klik foto Ahmad Nafis yang berdiri sebanyak 5x.
6. Login admin memakai passcode default: NafaPorto2026!
7. Edit tulisan/foto/kotak/section/layout, lalu klik Simpan ke Supabase.

Catatan:
- Website tetap single file index.html dan bisa dideploy sebagai static site di Vercel.
- Jangan upload file SQL ke public repo kalau tidak ingin passcode default terlihat. SQL cukup dijalankan di Supabase.
- Kalau ingin ganti passcode, jalankan ulang SQL dengan mengganti teks NafaPorto2026! di bagian insert portfolio_admin_secrets.
